# NR-Nightlord-Checker

## ✨ 소개
엘든 링 밤의 통치자의 깊은 밤 컨텐츠에서 마지막 날이 되기 전, **목표한 밤의 왕**과 **영원한 밤의 왕 여부**를 확인할 수 있는 프로그램입니다.



## ⚙️ 사용 방법
1. 엘든 링 밤의 통치자의 세이브 파일 위치에 실행 프로그램(exe)을 위치시킵니다.
   
   세이브 파일(NR0000.sl2) 위치: ```C:\%appdata%\Roaming\Nightreign\<SteamID>\```

2. 게임 실행 및 출격 후 맵 로딩이 완료된 상태에서 프로그램을 실행하여 목표 타겟을 확인합니다.
